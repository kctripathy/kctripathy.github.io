Current Project:

Project: EngagementCloudCore

EngagementCloudCore (formerly dotMailer), a comprehensive .NET-based digital marketing and customer engagement platform. Here's what this project encompasses:

This is a multi-channel marketing automation platform that provides:
• Email Marketing: Campaign creation, templates, personalization, and analytics
• Transactional Email: API-driven email delivery for applications
• SMS/MMS Marketing: Mobile messaging campaigns
• WhatsApp Business: WhatsApp messaging integration
• Web Chat: Live customer support chat functionality
• E-commerce Integration: Product recommendations, abandoned cart recovery, order tracking
• Marketing Automation: Triggered campaigns, customer journey workflows
• Contact Management: Segmentation, data management, and analytics
• Surveys & Forms: Data collection and lead generation tools

Architecture & Technology Stack
Multi-Target Framework:
• .NET Framework 4.8 (legacy components)
• .NET Standard 2.0 (shared libraries)
• .NET 8 (modern components)
Key Technologies:
• Backend: ASP.NET Web Forms, ASP.NET Core, C#
• Frontend: AngularJS/Angular, JavaScript/TypeScript
• Data: SQL Server, BigQuery (analytics), Azure Blob Storage
• Cloud: Azure, Google Cloud Platform
• Data Access: Dapper ORM, stored procedures
